# 🎵 LyriCord

**LyriCord** is a simple Python script that automatically updates your Discord status to display the lyrics of the song you choose. This project is based on [Lyrify](https://github.com/newst4rt/Lyrify).

---

## 🚀 Features
* **Real-time Sync:** Song lyrics update on your profile automatically.
* **Lightweight:** Minimal CPU and RAM usage.
* **Easy Setup:** Get it up and running in just a few minutes without complex tools.

---

## 🛠️ Installation & Setup

### 1. Download the Project
1. Click the green **Code** button at the top of this page.
2. Select **Download ZIP**.
3. Extract the downloaded ZIP file onto your computer.

### 2. Install Dependencies
Open your terminal (or Command Prompt) inside the extracted project folder and run:

pip install -r requirements.txt

### 3. Configure Your Token
1. Open the `main.py` file in any text editor (e.g., Notepad, VS Code).
2. Look for the `DISCORD_USER_TOKEN` variable.
3. Replace its value with your actual Discord token.

> 💡 **How to get your Discord Token?**  
> Check out this step-by-step [video tutorial on YouTube](https://www.youtube.com/watch?v=LnBnm_tZlyU) to find your token.

> [!WARNING]  
> **SECURITY WARNING:** Never share your Discord token with anyone! It grants full access to your account. Make sure you do not upload your `main.py` file containing your token back to public internet repositories.

### 4. Run the Script
To start displaying lyrics on your profile, run the following command in your terminal:

python main.py
